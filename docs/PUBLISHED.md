## 🚀 Próximos pasos

### 1️⃣ Clonar en otro equipo
```bash
git clone https://github.com/DanielProgram14/serkan-spa.git
cd serkan-spa
```

### 2️⃣ Instalar y ejecutar
**Backend**:
```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows: o source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

**Frontend**:
```bash
cd frontend
npm install
npm run dev
```

### 3️⃣ Listo para usar
- Backend en: http://localhost:8000
- Frontend en: http://localhost:5173
- Admin en: http://localhost:8000/admin
